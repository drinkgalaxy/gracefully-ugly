# 곱게 못난이, gracefully-ugly
### 못난이 농산물 공동구매 사이트
- 비싼 농산물 가격 + 못났다는 이유로 폐기되는 못난이 농산물 두 가지 문제를 해결하기 위해 못난이 농산물 공동구매 사이트 제작
- 판매자는 원하는 공동구매 인원을 모집할 수 있고, 구매자는 공동구매에 자유롭게 참여할 수 있음
- 이스트소프트의 앨런 AI 기능을 사용해 메인 페이지에서 최신 뉴스와 날씨 정보 확인 가능
